About Vertex Cover

A vertex cover of a graph is a set of vertices such that each edge of the graph is incident to at least one vertex in the set.

The minimum vertex cover problem is to find a vertex cover of minimum size.

This simulator uses a brute force approach to find the minimum vertex cover by checking all possible subsets of vertices.
